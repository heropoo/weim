# WeIM

### 运行websocket服务
1. 在终端执行 
```sh
php moon ws-server::run
```
    
2. web服务器入口到 `public/index.php`。请修改 `config/ws.php` 中的 `server` 项


### 开发环境
* PHP >= 7.0
* [php-swoole扩展](https://github.com/swoole/swoole-src)
* [Moon框架](https://github.com/heropoo/moon-app)
